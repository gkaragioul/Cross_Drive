OFFER FOR SOURCE CODE - GPL-COVERED COMPONENTS
==============================================

CrossDrive binary distributions may include object-code forms of GPL-covered
components, including:

- a custom WSL2 Linux kernel binary: wsl_kernel
- Linux filesystem kernel modules: hfsplus.ko, hfs.ko, apfs.ko
- optional APFS-FUSE-related binaries, if a particular release explicitly
  includes them

Recipients of GPL-covered binaries are entitled to the complete corresponding
source code for those GPL-covered portions under the terms of the applicable
GPL license.

How to Obtain Source
--------------------

1. The official CrossDrive source repository is:

       https://github.com/gkaragioul/Cross_Drive

   It contains a GPL source manifest at:

       docs/GPL_SOURCE_MANIFEST.md

   That manifest records the exact upstream source revisions, configuration,
   patches, and build scripts used for every GPL-covered binary included in
   each release.

2. If the corresponding source for a GPL-covered binary you received is not
   already present in the repository above, request it by opening an issue at:

       https://github.com/gkaragioul/Cross_Drive/issues

   The request will be fulfilled by providing a durable download link, a
   source archive, or a public repository containing the complete
   corresponding source.

3. Any fee charged for physical distribution of source on a medium will not
   exceed the reasonable cost of physically performing source distribution,
   in line with the applicable GPL license terms.

Duration
--------

This offer is valid for at least three years from the date of distribution
of the corresponding binary, and for as long as CrossDrive offers support
or spare parts for that release, whichever is longer. This offer is valid to
any third party in receipt of the binary, regardless of how they obtained it.
